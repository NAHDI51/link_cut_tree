Welcome to the **Link Cut Tree** repository! This project contains all the raw assets, animation scripts, and code implementations used to create an interactive and highly visual slide presentation illustrating the complex workings of a Link Cut Tree data structure.

## About The Project

Link Cut Trees are advanced data structures used to solve dynamic tree problems, but their internal mechanics (like `access`, `link`, `cut`, and splay operations) can be notoriously difficult to understand. 

The purpose of this repository is to break down these operations visually. By combining algorithmic implementations with smooth animations, this project serves as a comprehensive educational tool for understanding Link-Cut Trees.

## Features
* **Visualizations & Animations**: Python-based scripts to generate step-by-step animations of Link Cut Tree operations.
* **Algorithm Implementation**: Correct and optimized C++ implementations of the Link Cut Tree data structure.
* **Presentation Assets**: Ready-to-use assets for educational slides and presentations.

## Directory Structure

* 📁 **`assets/`** - Contains all the raw images, SVGs, and media files used in the slides and animations.
* 📁 **`build/`** - Output directory for generated presentation files and rendered video animations.
* 📁 **`implementation/`** - Contains the core C++ algorithms and logic for the Link-Cut Tree.
* 📁 **`scripts/`** - Python scripts used to generate the animations and process the visual components.

## 🛠️ Tech Stack
* **Python**: Used heavily (80%) for scripting the animations and visualizations.
* **C++**: Used (20%) for the underlying, performant implementation of the data structure.
